# Intelligent_recognition_shale-laminae
# Shale laminae are key geological features for sedimentary environment evolution and reservoir evaluation. Traditional manual identification suffers from low efficiency, high subjectivity, and poor scalability. To address the challenges of small thickness, blurred boundaries, subtle inter-class differences, and the original YOLOv8n model's susceptibility to missed detections, false detections, and insufficient feature focus, this paper presents an improved lightweight model named YOLOv8_c for intelligent laminae classification in shale thin sections.
